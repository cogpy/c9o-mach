# QEMU Boot Configuration Examples

Example QEMU configurations for booting and testing c9o-mach.
Inspired by: guile-daemon-zero's example configurations.

## Configurations

- `basic-i686.sh` - Minimal i686 boot for quick testing
- `debug-i686.sh` - Debug build with GDB stub enabled
- `smp-x86_64.sh` - Multi-processor x86_64 configuration
